# **Проектная работа по "Основы работы с данными для ИИ" Южного Федерального Университета**

Данный проект посвящён анализу датасета, содержащего информацию о трендах зимней моды за период 2023–2025 годов.
Целью исследования является выявление ключевых закономерностей, популярных категорий одежды, цветовых решений, материалов, а также изменений потребительских предпочтений во времени.

### **Цели и задачи**
  1. Исследование исходного датасета
  2. Предобработка данных (Удаление null значений и дубликатов)
  3. Анализ полученного датасета
  4. Визуализация закономерностей

### **Стек технологий**
  1. Jupyter Notebook
  2. Yandex Datalens
  3. Pyhton
  4. Pandas
  5. NumPy
  6. Matplotlib
  7. Seaborn
